Annotate & Markup

Add text, highlights, lines, images and black-out boxes on top of your PDF.

This tool adds text, shapes and markup on top of the PDF. It can't rewrite the document's existing body text — PDFs aren't built for that. To change the original wording, edit the source file (e.g. Word) and export a fresh PDF.
